# Chirp REST API

## Description

Made with [NestJS](https://github.com/nestjs/nest) framework.

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Making migrations

```bash
# generate migration
npm run typeorm migration:create <migration-name>

# run migrations
npm run migration:run

# revert migrations
npm run migration:revert
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
